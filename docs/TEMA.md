# Tema LabDev Dark

Tema oscuro con acentos fucsia para LabDev Editor.

## Colores

| Elemento | Color | Hex |
|----------|-------|-----|
| Keywords | Fucsia | #ff00ff |
| Variables | Fucsia | #ff00ff |
| Cursores | Fucsia | #ff00ff |
| Bordes activos | Fucsia | #ff00ff |
| Botones | Fucsia | #ff00ff |
| Fondo editor | Negro | #1e1e1e |
| Fondo sidebar | Negro | #1a1a1a |
| Texto | Gris claro | #d4d4d4 |

## Activar el Tema

1. Presiona `Ctrl+K` luego `Ctrl+T`
2. Selecciona "LabDev Dark"

## Personalizar

Edita el archivo:
```
LabDev-Editor\resources\app\extensions\theme-defaults\themes\labdev-dark-color-theme.json
```

Busca `#ff00ff` y reemplaza por tu color preferido.
